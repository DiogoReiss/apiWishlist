# WishList

The purpose of this simple Api is to manage a wish-list, adding products, removing and list the wish-list

## Installation

I often use yarn to manage the packages but you can use as well npm

```bash
yarn 
```

or 

```bash
npm install -i
```

## Usage

If u want to develop in this project, you only need to write a few commands
```bash
yarn knex:migrate
```
to make the data base

and
```bash
yarn develop
```
to start the server
